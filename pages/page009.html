Recursos para Apoiar a Jornada de Migração para a Nuvem

Explore ferramentas, serviços e boas práticas oferecidas pela AWS para facilitar a transição para a nuvem.

1. Planejamento e Avaliação

  • AWS Migration Hub: Centraliza o gerenciamento de migração, permitindo rastrear o progresso de aplicativos em várias ferramentas.
  • AWS Migration Readiness Assessment (MRA): Avalia a prontidão para a migração e fornece orientações sobre lacunas e pontos de melhoria.
  • AWS Well-Architected Framework: Oferece boas práticas para projetar cargas de trabalho confiáveis, eficientes e seguras na nuvem.

2. Descoberta e Planejamento

  • AWS Application Discovery Service: Automatiza a descoberta de aplicativos e dependências no ambiente local.
  • AWS Total Cost of Ownership (TCO) Calculator: Estima os custos de operação na nuvem comparados com ambientes locais.

3. Execução da Migração

  • AWS Server Migration Service (SMS): Migra servidores físicos e virtuais em larga escala para a AWS.
  • AWS Database Migration Service (DMS): Migra bancos de dados relacionais, não relacionais e data warehouses para a AWS.
  • AWS Snow Family: Inclui dispositivos como Snowball e Snowcone para transferir grandes volumes de dados.
  • AWS Transfer Family: Transferências seguras de arquivos via SFTP, FTPS e FTP diretamente para o S3.

4. Otimização Pós-Migração

  • AWS Cost Explorer: Oferece insights sobre os custos e uso da AWS.
  • AWS Trusted Advisor: Avalia a infraestrutura e fornece recomendações para otimização.
  • Amazon CloudWatch: Monitora métricas e logs para identificar gargalos e melhorar o desempenho.

5. Programas e Suporte

  • AWS Migration Acceleration Program (MAP): Programa que fornece recursos técnicos e financeiros para migrações em larga escala.
  • AWS Training and Certification: Cursos e certificações para preparar equipes para projetar e operar na AWS.
  • AWS Partner Network (APN): Rede de parceiros qualificados para facilitar a migração.
